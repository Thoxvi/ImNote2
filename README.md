## 介绍
 * 它是一个十分方便、小巧而又不失强大的卡片记录工具
 * 界面美观，使用了Google推荐的`Material Design`设计
 * 删除、创建十分方便，长按`+`和长按`清空回收站`Fab都有独特功能
 * 卡片提醒可以让你在关闭APP的时候及时看到提醒，设置权限之后甚至可以在锁屏界面查看
 * 独特的剪贴板监控功能，让你不用打开APP，直接复制文字，就会自动出现提示框让你新建
 * 体积小，只有2M的安装包，方便传输安装
 * 图片暂时没有截……
 * 可以试着下载编译一下玩玩！
